montecarlo.metropolis_sample(n=8, mu=1.1, k=1, J=-2, T=1, sweeps=100)[source]¶

Computes the approximate values of average energy, average magnetization, heat capacity, and magnetic susceptibility of an arbitrary spin lattice of size n using the montecarlo simulation

Parameters

  • n – The size of the lattice, defaults to 8

  • mu (float or int) – the magnetic constant, defualts to 1.1

  • k (float or int) – the boltzmann constant, defaults to 1

  • J (float or int) – the ferromagnetic constant, defaults to -2

  • T (float or int) – the temperature your state is in, defaults to 1

  • sweeps (int) – The number of sweeps you’ll take (more sweeps means more accurate simulation), defaults to 100

Returns

A list, the first item is the average energy, the second is the average magnetization, the third is the heat capacity, the fourth is the magnetic susceptibility

Return type

list
