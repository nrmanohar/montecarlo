class montecarlo.Hamiltonian(J=-2, mu=1.1, k=1, state=[1, -1, 1, -1])[source]¶

This is a simple class to calculate the hamiltonian for a spin configuration

Parameters

  • J (int, optional) – The ferromagnetic constant for the material, defaults to -2

  • mu (int, optional) – magnetic constant for energy calcualation, defaults to 1.1

  • k (float, optional) – The Boltzmann constant, defaults to 1

  • state (list, optional) – The state that is being analyzed, defaults to [1,-1,1,-1]
